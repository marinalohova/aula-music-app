# aula-music-app
To run Aula Music App:

Start the backend
#### `cd aula-music-service`
#### `nvm use 11.12.0`
#### `npm i`
#### `npm start`

Start the frontend
#### `cd aula-music-client`
#### `nvm use 11.12.0`
#### `npm i`
#### `npm start`

Open http://localhost:3000 in your browser. 
Please, refer to the individual directory README files for more details.
